# Настройка Vercel KV для базы данных

## Шаги настройки:

1. **Создайте KV базу данных в Vercel:**
   - Перейдите в ваш проект на Vercel
   - Откройте вкладку "Storage"
   - Нажмите "Create Database"
   - Выберите "KV" (Redis)
   - Создайте базу данных

2. **Получите переменные окружения:**
   После создания KV базы данных, Vercel автоматически добавит переменные окружения:
   - `KV_REST_API_URL`
   - `KV_REST_API_TOKEN`

3. **Проверьте переменные окружения:**
   - Перейдите в Settings → Environment Variables
   - Убедитесь, что `KV_REST_API_URL` и `KV_REST_API_TOKEN` установлены

4. **Перезапустите деплой:**
   После добавления переменных окружения, перезапустите деплой вашего приложения.

## Как это работает:

- **На проде (Vercel):** Данные сохраняются в Vercel KV (Redis)
- **Локально:** Данные сохраняются в файл `data/subjects.json` (для разработки)
- **Автоматическая инициализация:** При первом запуске на проде, данные из файла автоматически загружаются в KV

## Примечание:

Если переменные окружения KV не установлены, приложение будет использовать файлы (как раньше), но на проде это не будет работать из-за read-only файловой системы. Поэтому обязательно настройте KV для продакшена.
